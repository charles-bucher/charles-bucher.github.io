## Overview
This portfolio showcases:

- Automated monitoring pipelines using Python, AWS CloudWatch, and Lambda  
- Proactive alerting workflows for operational incident response  
- CI/CD automation for scalable AWS environments  
- Simulated operational troubleshooting and cloud incident resolution  

**Tech stack & tools:** Python, AWS (CloudWatch, Lambda, SNS, EC2, S3, ECS), Terraform, GitHub Actions, CI/CD, Serverless, Logging & Observability

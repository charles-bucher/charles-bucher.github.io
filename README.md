### 2️⃣ Multi-Tier Application Troubleshooting
**Troubleshooting a broken 3-tier web application (Frontend → Backend → Database)**

**Problems Diagnosed & Fixed:**
- Load balancer health checks failing
- Database connection timeouts
- API endpoints returning 500 errors
- Network routing issues between tiers

**Troubleshooting Process:**
1. **Gather info:** Check CloudWatch logs, instance status, security groups
2. **Isolate issue:** Test each tier independently
3. **Identify root cause:** Document findings with screenshots
4. **Implement fix:** Apply configuration changes
5. **Verify resolution:** Confirm service restored

**Skills Demonstrated:**
- Multi-layer troubleshooting
- Reading application logs
- Understanding network flows
- Following a structured support process

**Tech Used:** AWS (EC2, RDS, VPC, ELB, CloudWatch), Terraform for infrastructure

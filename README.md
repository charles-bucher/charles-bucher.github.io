## Overview

This portfolio highlights hands-on CloudOps, AWS, and NOC skills. It demonstrates:

- Automated monitoring pipelines using Python, AWS CloudWatch, and Lambda  
- Proactive alerting and incident response workflows  
- CI/CD automation for scalable AWS environments  
- Multi-tier application troubleshooting and operational simulations  

**Tech stack & tools:** Python, AWS (EC2, CloudWatch, Lambda, SNS, S3, ECS), Terraform, GitHub Actions, CI/CD, Serverless, Logging & Observability

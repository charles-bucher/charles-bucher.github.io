## 🚀 Running Locally

This site is built with Jekyll and GitHub Pages.

```bash
# Clone repository
git clone https://github.com/charles-bucher/charles-bucher.github.io.git
cd charles-bucher.github.io

# Install dependencies (if using Jekyll locally)
bundle install

# Run local server
bundle exec jekyll serve

# View at http://localhost:4000
```
